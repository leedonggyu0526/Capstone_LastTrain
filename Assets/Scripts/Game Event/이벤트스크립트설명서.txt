EventData.cs	
데이터 구조 정의. EventData.csv의 컬럼(eventID, eventName, effectType_1 등)과 1:1로 매칭되는 필드 및 속성을 가진 C# 클래스 또는 구조체입니다. CSV 데이터를 메모리에 로드할 때 사용됩니다.
EventDatabase.cs	
데이터 관리자. EventData.csv 파일을 읽어 파싱하고, 모든 EventData 객체들을 Dictionary<int, EventData> 형태로 저장하는 중앙 저장소입니다. EventSpawner가 이벤트 ID를 통해 데이터를 요청할 때 이 클래스를 사용합니다.
EventEffectType.cs	
상수 정의. CSV에서 사용된 효과 유형(DAMAGE_DURATION, DEBUFF_RATE) 및 대상(TRAIN_HP, ENGINE) 문자열을 enum 또는 const string으로 정의하여, 코드에서 발생할 수 있는 오타로 인한 런타임 오류를 방지합니다.
EventSpawner.cs	
데이터 연결 및 전달. TrySpawnEvent() 함수가 다음 로직을 수행하도록 수정해야 합니다: 1) EventDatabase.cs에 접근하여 랜덤으로 선택된 eventID에 해당하는 EventData를 가져옵니다. 2) 선택된 EventObject의 ActivateEvent()를 호출할 때, 이 EventData 객체를 인수로 전달합니다.
EventObject.cs	
데이터 수신 및 표시. public void ActivateEvent() 메서드를 public void ActivateEvent(EventData data) 형태로 변경해야 합니다. 전달받은 EventData의 eventName 등을 이벤트 UI 컴포넌트(Text, Image 등)에 할당하여 화면에 표시하는 로직이 추가되어야 합니다.
